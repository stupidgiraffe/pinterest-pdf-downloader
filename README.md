# Pinterest to PDF Downloader 📌📄

A lightweight userscript that allows you to download Pinterest images and posts directly as high-quality PDFs. 

## 🚀 Installation
1. Install a userscript manager like [Tampermonkey](https://www.tampermonkey.net/), [Violentmonkey](https://violentmonkey.github.io/), or [ScriptCat](https://scriptcat.org/).
2. Click [HERE](https://github.com/stupidgiraffe/pinterest-pdf-downloader/raw/main/pinterest-downloader.user.js) to install the script directly.
